# High-Fidelity UI Design

## Overview

The Smart Online Shopping application was designed using modern UI/UX principles in Figma. The interface uses consistent colors, typography, icons, and spacing to provide an intuitive shopping experience.

## Screens Included

1. Login
2. Home
3. Product 
4. add to Cart
5. cart
6. Checkout
7. Order Confirmation
8. Profile

## Design Features

- Consistent color palette
- Responsive mobile layout
- Modern typography
- User-friendly navigation
- High-quality product images
- Clear call-to-action buttons
