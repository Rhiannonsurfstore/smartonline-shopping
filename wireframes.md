# Wireframes

## Overview
These wireframes show the basic structure of the Smart Online Shopping application. They focus on layout and user interface structure without colors or final styling.

---

## Wireframe Screens Included

### 1. Login Page
- Email input field
- Password field
- Login button
- Forgot password option

### 2. Home Page
- Search bar
- Product categories
- Product grid layout

### 3. Product Details Page
- Product image placeholder
- Product name and price
- Description section
- Add to cart button

### 4. Cart Page
- List of selected items
- Total price
- Checkout button
